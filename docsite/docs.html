Usage

Add this library by including the following CSS and JavaScript files from CDN. You can also import individual components if you only need a few of them. The links to import individual components can be found in their respective pages.

Alert

Alerts are used to attract user's attention for important information without interrupting the user's flow. The alert component has four variants:-

A simple success alert.
A simple failure alert.
A simple dark alert.
A simple light alert.
